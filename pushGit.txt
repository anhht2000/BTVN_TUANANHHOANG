//git reset
Khi mà b push code lên rồi mới phát hiện ra bị lỗi mà muốn trở về cái cũ(xóa cái hiện tại):

git reset --hard idCommit(cũ) (ở local, nó sẽ xóa đi các commit sau idCommit)
muốn lấy idcommit thì git log ra
git push -f (bắt buộc để ghi đè những cái commit trên remote)

//git revert(Commended)
Khi mà b push code lên rồi mới phát hiện ra bị lỗi mà muốn trở về cái cũ(đổi về cái cũ)
git revert idCommit: tạo ra commit mới giống hệt cái idCommit để đè lên cái commit hiện tại
cũng phải push lên
